# Database Management System 資料庫

## 作業目標
作業須具備：瀏覽、查詢、刪除和新增功能。

## 實行想法
我的主題是介紹貓咪品種的網站，包含貓咪含有哪些天生疾病，以及全台各地的動物醫院可以供瀏覽者查詢。在瀏覽者頁面中可以對貓咪品種進行搜尋，以及對動物醫院的所在地進行查詢；在網站管理者頁面中亦可以對貓咪品種進行搜尋，除此之外可以新增、刪除貓咪及動物醫院的各項資訊，方便他們做網頁的更新。
